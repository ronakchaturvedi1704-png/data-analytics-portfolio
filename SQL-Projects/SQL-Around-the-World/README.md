# SQL Around the World Analysis

## Overview
This project analyzes a global dataset using SQL to explore country-level data and identify patterns and trends across regions.

## Dataset
The dataset contains information related to different countries, including geographic and statistical attributes.

## Tools Used
- SQL

## Analysis Performed
- Filtered records to focus on specific regions and criteria
- Grouped data to compare countries and regions
- Used aggregation functions to summarize key metrics
- Sorted results to identify top and bottom values

## Key Insights
- Significant differences exist between regions across key metrics
- Certain countries consistently stand out within their regions
- Aggregated data helps reveal global patterns that are not visible at the individual record level
